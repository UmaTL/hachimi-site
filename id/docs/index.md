# Dokumentasi

Selamat datang di halaman dokumentasi Hachimi Edge!
Lihat semua topik yang tersedia di bilah sisi (sidebar) sebelah kiri. Untuk pengguna perangkat seluler, kamu bisa menemukannya di menu pada bagian kiri atas.

## Memasang Hachimi

Silakan cek halaman [Memulai](hachimi/getting-started).

## Penggunaan Plugin

Hachimi Edge mendukung [sistem plugin](plugins/about) agar bisa memperluas fungsionalnya.

## Berkontribusi Sebagai Translator

Hachimi Edge membutuhkan penerjemah untuk UI, situs/dokumentasi, dan dukungan pengguna. Jika kamu ingin membantu, silakan bergabung ke [Discord](https://discord.gg/hachimimod) dan panggil/mention `Developer`.
Untuk situs ini, kamu bisa berkontribusi langsung ke [repositori GitHub](https://github.com/UmaTL/hachimi-site).

Untuk membantu terjemahan game, silakan pelajari [Panduan Translasi](translation-guide/welcome) dan hubungi [salah satu kontributor proyek](/id/credits).

## Buat Sumber Translasimu Sendiri

Setelah kamu memahami [Panduan Translasi](translation-guide/welcome), pelajari cara [konfigurasi repo](translation-guide/repo-config) dan kirimkan proyek kamu ke indeks [Hachimi Edge](https://github.com/UmaTL/hachimi-meta). Kamu bisa melihat [salah satu proyek](/id/credits) yang sudah ada sebagai referensi.
