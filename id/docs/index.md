## Berkontribusi

Silakan baca dokumen terkait dan periksa berbagai repositori Git yang ada. Kamu bisa membuka *Issue* di sana atau langsung mengirimkan PR (*Pull Request*),  dalam beberapa kasus disarankan untuk mendiskusikan hal tersebut terlebih dahulu dengan tim.
Jika kamu membutuhkan bantuan lebih lanjut atau ingin menghubungi kami secara langsung, bergabunglah dengan [server Discord](https://discord.gg/hachimimod) dan sebut (*mention*) *role* `Developer` di saluran `#feedback`

### Sebagai pengembang Hachimi Edge

Pengembangan dilakukan di [repo Hachimi Edge](https://github.com/kairusds/Hachimi-Edge), dengan diskusi paling aktif di Discord.

### Sebagai penerjemah proyek Hachimi

Berkaitan dengan penerjemahan UI Hachimi Edge, situs & dokumentasi, pengumuman komunitas, dan dukungan pengguna. Semuanya untuk bahasa yang non-Inggris.
Kontribusi langsung ke [repositori GitHub](https://github.com/UmaTL/hachimi-site) atau gabung Discord untuk menanyakan perihal penyediaan translasi bagi komunitas.

### Sebagai penerjemah game

Proyek Hachimi tidak menangani translasi game itu sendiri.
Jika kamu ingin membantu, silakan pelajari [Panduan Translasi](translation-guide/welcome) dan hubungi [salah satu kontributor proyek](/id/credits).
Bergabung di Discord juga berguna untuk mendapatkan informasi terbaru atau mengajukan pertanyaan teknis. Hubungi salah satu `Server Managers` untuk mendapatkan *role*.

#### Membuat sumber translasimu sendiri

Tidak ada proyek untuk bahasamu atau tidak setuju dengan yang sudah ada? Mulai proyekmu sendiri!
Setelah kamu memahami [Panduan Translasi](translation-guide/welcome), pelajari cara [konfigurasi repo](translation-guide/repo-config) dan kirimkan proyek kamu ke [index Hachimi Edge](https://github.com/UmaTL/hachimi-meta).
Kamu bisa melihat [salah satu proyek](/id/credits) yang sudah ada sebagai referensi.

**Catatan**: Kami sarankan untuk tetap menggunakan satu proyek per bahasa jika memungkinkan guna menghindari fragmentasi (perpecahan) atau kerja dua kali. Jika tidak, pastikan kamu mematuhi ketentuan proyek yang sudah ada, jangan melakukan *fork* secara asal.

## Sebagai relawan komunitas

Discord Hachimi Edge membutuhkan orang-orang untuk membantu pengguna lain dengan masalah teknis, menyediakan pengujian bug dan fitur bagi para pengembang, dll.
Bergabunglah dengan Discord dan ajukan pendaftaran di saluran `#applications-info`.
