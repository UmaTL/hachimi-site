# Selamat Datang
Selamat datang di panduan translasi Hachimi! panduan ini akan memandu kamu ke sistem translasi Hachimi, cara kerja translasi, dan bagaimana cara menggunakan alat translasi ZokuZoku.
## Glosarium/Istilah
Sebelum membaca panduan, Pastikan kamu membiasakan diri dengan beberapa istilah yang akan digunakan di seluruh artikel.
- **Dict (_Dictionary_):** Digunakan untuk merujuk pada file terjemahan yang berfungsi layaknya kamus, sistem terjemahan akan menelusuri entri dalam file tersebut untuk mendapatkan terjemahan yang sesuai.
- **Repo (_Repository_):** Jika kamu belum pernah mendengar istilah ini dalam konteks [_Version control_](https://en.wikipedia.org/wiki/Version_control), Kamu bisa dengan mudah menafsirkannya sebagai “tempat di mana terjemahan disimpan”. Ketika istilah ini digunakan, biasanya merujuk pada dua jenis repo: _Git repository_ (digunakan untuk _version control_/kolaborasi) atau _TL repository_ (digunakan oleh Hachimi untuk pembaruan terjemahan). Yang pertama adalah yang paling umum disebut, sedangkan yang kedua lebih banyak berkaitan dengan pengelola repo dan akan dijelaskan lebih mendalam dalam artikelnya sendiri.
- **Meta (Metadata):** Informasi tambahan untuk sesuatu. Dalam konteks kamus terjemahan Hachimi, biasanya merujuk pada entri khusus di dalam kamus untuk file aset yang menentukan bundle hash-nya.
- **TL:** Singkatan dari "translasi".
- **Dir (_Directory_):** Istilah elegan untuk menyebut
 "folder".
- **ZokuZoku:** Ini adalah nama dari alat translasi utama Hachimi. "_Juga merupakan lagu yang dinyanyikan Transcend_".
