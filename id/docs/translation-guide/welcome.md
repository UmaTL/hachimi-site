# Selamat Datang

Selamat datang di panduan translasi Hachimi! panduan ini akan memandu kamu ke sistem translasi Hachimi, cara kerja translasi, dan bagaimana cara menggunakan alat translasi ZokuZoku.

## Glosarium/Istilah

Sebelum membaca panduan, Pastikan kamu membiasakan diri dengan beberapa istilah yang akan digunakan di seluruh artikel.

- **Dict (*Dictionary*):** Digunakan untuk merujuk pada file terjemahan yang berfungsi layaknya kamus, sistem terjemahan akan menelusuri entri dalam file tersebut untuk mendapatkan terjemahan yang sesuai.
- **Repo (*Repository*):** Jika kamu belum pernah mendengar istilah ini dalam konteks [*Version control*](https://en.wikipedia.org/wiki/Version_control), Kamu bisa dengan mudah menafsirkannya sebagai “tempat di mana terjemahan disimpan”. Ketika istilah ini digunakan, biasanya merujuk pada dua jenis repo: *Git repository* (digunakan untuk *version control*/kolaborasi) atau *TL repository* (digunakan oleh Hachimi untuk pembaruan terjemahan). Yang pertama adalah yang paling umum disebut, sedangkan yang kedua lebih banyak berkaitan dengan pengelola repo dan akan dijelaskan lebih mendalam dalam artikelnya sendiri.
- **Meta (Metadata):** Informasi tambahan untuk sesuatu. Dalam konteks kamus terjemahan Hachimi, biasanya merujuk pada entri khusus di dalam kamus untuk file aset yang menentukan bundle hash-nya.
- **TL:** Singkatan dari "translasi".
- **Dir (*Directory*):** Istilah elegan untuk menyebut
 "folder".
- **ZokuZoku:** Ini adalah nama dari alat translasi utama Hachimi. "*Juga merupakan lagu yang dinyanyikan Transcend*".
