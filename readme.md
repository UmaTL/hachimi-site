# Hachimi Edge site

The repo for the Hachimi Edge website, hosting guides and documentation.

## Contributing

Create PRs directly or ask questions through Issues or the [Hachimi Project Discord](https://discord.gg/hachimimod).

Please use markdownlint through [markdownlint-cli2](https://github.com/DavidAnson/markdownlint-cli2) or [the vscode extension](https://marketplace.visualstudio.com/items?itemName=DavidAnson.vscode-markdownlint) to verify your MD files. A custom config is provided.
