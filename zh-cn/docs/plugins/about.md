# 插件
Hachimi 支持插件系统，允许开发者通过动态库扩展其功能。插件可以 Hook 游戏、修改行为、添加 GUI 元素等。

## 什么是插件？

插件是与 Hachimi Edge 通过预定义 API 集成的动态库（Android 上为 `.so` 文件，Windows 上为 `.dll` 文件）。它们与游戏在同一进程中运行，并拥有修改游戏行为的强大功能。

## 插件能做什么？

插件可以访问全面的 API，从而实现：

* **Hook 并检查 IL2CPP**：拦截函数并访问 Unity IL2CPP 的类、方法和字段。
* **添加 GUI 元素**：在 Hachimi 内置 GUI 中注册菜单项、板块并显示通知。
* **Android DEX 加载**：在 Android（API v2+）上加载并执行 Java/Kotlin 代码。
* **……以及更多功能。**

## 安全注意事项 ⚠️

切勿安装来自未知来源或不受信任开发者的插件。
插件在运行时拥有游戏进程的完整访问权限。恶意插件可能会：

* 窃取您的游戏账号凭据和会话令牌。
* 修改游戏数据，可能导致账号被封禁。
* 导致游戏崩溃或数据损坏。
* 将您的游戏数据发送到外部服务器。
* 在游戏的权限范围内执行任意代码。

::: danger ⚠️ 重要安全警告
**在 Windows 平台上，由于游戏以管理员权限运行，插件拥有对您整个系统的“完全管理员访问权限”。它们可以安装恶意软件、修改系统文件、访问您的所有数据等。**
:::

在安装插件之前，请考虑：

* **来源信任**：仅使用来自您信任的开发者的插件。
* **社区验证**：检查该插件是否经过社区审核。
* **更新风险**：曾经信任的插件可能会随着时间推移而变得具有恶意。

Hachimi Edge 的开发者无法验证第三方插件，且不对您因使用插件而导致的任何问题或损失负责。
